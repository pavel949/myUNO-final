# legacy — your existing code (reference only)

Put each existing repository here, one subfolder each. Clone (`git clone <url> ./legacy/<name>`) or copy in.

**A parts bin, not a foundation, and not the look.** Fable reads them in document 0 and makes a **take / don't-take / finish** decision on every reusable piece. The platform is built fresh around **project → unit → identity → roles**; the visual language is single and new.
