# myUNO — Brand & Art Direction

The **art direction**. Fable expands it into a full design system (`docs/06_design_system.md`). Agents build UI only from that system. Visual references: `myuno_brand.html`, `myuno_positioning.html`.

## Personality
Warm, but trustworthy. Personal, but institutional. Simple, but premium. Calm confidence. Never loud, never tropical-kitsch, never a cheap startup look.

## Name & wordmark
**myUNO** — one word. “my” lighter and recessed; “UNO” solid in Andaman teal (UNO = one). The final **O** is the mark. Lowercase `myuno` only for domains.

## The mark
A ring with a centre point: one / whole, you at the centre (gold point = the person), a quiet echo of the loop.

## Colour
Andaman `#0E4F4B` · Deep `#0A3733` · Sun `#D69A3A` · Ivory `#F5EFE4` · Ink `#16211F`, plus muted stone and functional states tuned to the palette.

## Typography
Display / wordmark: Outfit. Body: Manrope.

## Voice
Calm, plain, warm. “One” runs through it. “Your stay, sorted.” · “Real, and verified.” · “Your one place.”

## Across projects
Endorsed, not absorbed: the project leads (“Layan Verde”); myUNO is the layer underneath (“on myUNO”).

## Tagline
**Your one place.**

## Note
A proposal; flexes to any existing myUNO assets.
